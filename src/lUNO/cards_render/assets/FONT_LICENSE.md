# 同梱フォントについて

`DejaVuSans-Bold.ttf` は **DejaVu Fonts**（Bitstream Vera / Arev 由来）である。
カード画像生成（`cards_render/generator.py`）で環境差を消すため同梱している。

- プロジェクト: https://dejavu-fonts.github.io/
- ライセンス: DejaVu Fonts License（Bitstream Vera / Arev のパーミッシブ・ライセンス。
  再配布・改変・バンドルが許可される）。全文は上記プロジェクトの `LICENSE` を参照。

DejaVu は "DejaVu" 商標条項を除き自由に利用・再配布できる。本リポジトリはフォントを
そのまま同梱（無改変）しているため、この条項に抵触しない。
